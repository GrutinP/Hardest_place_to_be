# CSS雨演出 学習計画

目的: CSSの雨演出を自分で理解して作れるようにする。最終的にはティラノスクリプトに移植できる形まで進める。

## 方針

- 一度に進めない。
- 1ステップにつき、覚える概念は1つだけにする。
- 各ステップでブラウザプレビューを確認する。
- 仕組みが分かってから、ティラノスクリプトに入れる。

## 全体図

### 1. 雨粒1本を表示する

学ぶこと:

- HTMLの要素
- CSSの `width` / `height` / `background`
- 雨粒は「細い四角」だと理解する

ゴール:

- 画面に白い縦線が1本出る

### 2. 雨粒1本を好きな位置に置く

学ぶこと:

- `position: absolute`
- `left`
- `top`

ゴール:

- 雨粒を画面の好きな場所に置ける

### 3. 雨粒1本を下へ動かす

学ぶこと:

- `@keyframes`
- `animation`
- `from` / `to`

ゴール:

- 雨粒が上から下へ落ちる

### 4. 速度を変える

学ぶこと:

- `animation-duration`
- `linear`
- `infinite`

ゴール:

- 雨粒の落ちる速さを自分で調整できる

### 5. 雨粒を複数にする

学ぶこと:

- 同じCSSを複数の要素に使う
- `left` を変えて横に散らす

ゴール:

- 雨っぽく複数の線が落ちる

### 6. 落ちるタイミングをずらす

学ぶこと:

- `animation-delay`

ゴール:

- 全部同時に落ちず、自然に見える

### 7. 見た目を雨らしくする

学ぶこと:

- `opacity`
- `linear-gradient`
- `transform: rotate`

ゴール:

- 白い棒ではなく、雨粒らしく見える

### 8. 背景の上に重ねる

学ぶこと:

- 親要素
- `overflow: hidden`
- `z-index`

ゴール:

- 背景画像の上に雨だけ重ねられる

### 9. ティラノスクリプトに入れる

学ぶこと:

- どのレイヤーに重ねるか
- CSSをどこに置くか
- HTML要素をどう作るか

ゴール:

- ゲーム画面上で雨が降る
