# シナリオ構成

## あらすじ

テレビの星座占いで11位になったプレイヤーが、同じ帰り道の女の子（アジサイ）と一緒に駅まで歩く話。
プレイヤーが選択で自分を守るとアジサイがひどい目に遭い、アジサイを守るとプレイヤーがひどい目に遭う。

**仕掛け：** アジサイは「自分も同じ星座」と同調しているが、実は最下位の12位（魚座）。
ラッキーアイテム「赤い傘」を折りたたみ傘として鞄に持っていた。

---

## ファイル構成と分岐

```
00_init          初期化
01_avan          名前・星座入力
02_opening       テレビの星座占いコーナー
03_prologue      出会い・相合傘になる

04_bad01         【1つめの不幸】 ③◆ 水たまりのトラック
├── 05_bad01_a   ③A　道路から離れる → アジサイが水を被る
└── 06_bad01_b   ③B　道路に近づく   → プレイヤーが水を被る

        ↓ シナリオ合流

07_bad02         【2つめの不幸】 ⑥◆ 大型犬
├── 08_bad02_a   ⑥A　右に交わす    → 犬がアジサイに飛びかかる
└── 09_bad02_b   ⑥B　知り合いと呼ぶ → 犬がプレイヤーに飛びかかる

        ↓ シナリオ合流

10_bad03         【3つめの不幸】 ⑧◆ 傘が壊れる
├── 11_route_a   ⑧A　走る           → アジサイと走って駅へ
│                ⑨A 選択肢（影響なし）
│                → ⑨Aルート合流地点 → 20_gate_common
└── 12_route_b   ⑧B　コンビニへ
    ├── 13_route_b_green  緑の傘を買った → 20_gate_common
    └── 14_route_b_red    赤い傘を買った
        ├── 15_route_b_red_yes  ⑩◆「はい」 → アジサイの告白長文 → 17_quiz
        └── 16_route_b_red_no   ⑩◆「いいえ」 → あたふた      → 17_quiz

17_quiz
├── f.chara_saved == 0 かつ Bルート → 18_quiz_main（星座当てクイズ）
│   ├── 正解 → 25_true_end（トゥルーED）
│   └── 不正解 → 19_quiz_wrong → 24_ed_1（ED①）
└── それ以外 → 24_ed_1（ED①）

20_gate_common   改札後・別れ際
├── f.chara_saved == 3（全てアジサイがひどい目）→ 21_ed_a（EDa）
├── f.chara_saved == 0（全てプレイヤーが犠牲）  → 23_ed_c（EDc）
└── それ以外（1 or 2）                          → 22_ed_b（EDb）
```

## エンディング一覧

| ファイル | ED | 条件（20_gate_common / 17_quiz） | 内容 |
|---|---|---|---|
| 21_ed_a | EDa | chara_saved==3 | アジサイがプレイヤーを守り切れてよかったとはしゃぐ |
| 22_ed_b | EDb | chara_saved==1 or 2（混合） | 短い道のりが大冒険だった |
| 23_ed_c | EDc | chara_saved==0 | プレイヤーの優しさに気付いているアジサイ |
| 24_ed_1 | ED① | 赤の傘ルートで chara_saved>0、または星座当て失敗 | 赤い傘への想いを語る |
| 25_true_end | TRUE | 赤の傘ルートで chara_saved==0 かつ星座当て正解 | 無言・スチル |

**chara_saved の値域（③⑥⑧Aの選択のみ加算、最大3）:**
- ③A + ⑥A + ⑧A → 3（全てアジサイがひどい目 = EDa）
- その他の組み合わせ → 1 or 2（EDb）
- ③B + ⑥B + ⑧B → 0（全てプレイヤーが犠牲 = 自己犠牲コンプ → EDc or 星座クイズ挑戦）
- ※⑧Aを選ぶと最低1になるため chara_saved==0はBルート緑からのみ到達可

## 変数

### ゲーム進行・フラグ

| 変数名 | 型 | 初期値 | 用途 |
|---|---|---|---|
| `f.p_name` | string | `''` | プレイヤーの名前（01_avanで入力） |
| `f.p_sign` | number | `0` | プレイヤーの星座番号（1〜12） |
| `f.c_sign` | number | `12` | アジサイの星座番号（魚座=12。p_sign==12なら6に変更） |
| `f.chara_saved` | number | `0` | アジサイがひどい目にあった回数（最大3、ED分岐に使用） |
| `f.if_chara_splashed` | 0/1 | `0` | 水浸しパーツ表示（③A選択で1） |
| `f.if_chara_stomped` | 0/1 | `0` | 犬の足跡パーツ表示（⑥A選択で1） |
| `f.if_chara_stucked` | 0/1 | `0` | 葉・枝パーツ表示（3つめの不幸で1） |

### 魚座分岐制御変数（01_avanの星座選択後に一括設定）

| 変数名 | 型 | 魚座(p_sign==12) | それ以外 | 用途 |
|---|---|---|---|---|
| `f.c_name` | string | `"アジサイ"` | `"＊"` | 会話中の名前ボックス表示。`#&f.c_name` で使用 |
| `f.ch_reflect` | string | `"true"` | `"false"` | 立ち絵の左右反転。`[chara_show reflect="&f.ch_reflect"]` で使用 |
| `f.rain_deg` | number | `-5` | `5` | 雨アニメーションのdeg値。プレイヤーが魚座なら逆方向 |
| `f.walk_keyframe` | string | `"walk_bg_scroll_rev"` | `"walk_bg_scroll"` | 歩き背景のキーフレーム名。`[xanim keyframe="&f.walk_keyframe"]` で使用 |

※`f.chara_saved`（回数）と汚れパーツ変数は独立。どの汚れが乗っているかを個別に管理する。

| タイミング | 場面 | ファイル | 加算 |

## キャラ立ち絵レイヤー構造

立ち絵はレイヤーの重ね合わせで構成される。

```
【ベース】2種類（シーンに応じて切り替え）
  ├── 全身立ち絵     ADVパート用（太もも以上、ただし全身データ必要）
  └── 横歩き絵       道中シーン用（プレイヤーの左、画面右向き）

【顔パーツ】ベースの上に乗せる（眉・目・口の組み合わせ）
  ├── 眉：怒り / 困り / 通常
  ├── 目：通常 / 目閉じ / 目閉じ笑み / 驚き / ジト目
  └── 口：通常 / ▽（口開け笑い）/ イー / へ / 笑み / O / x（ミッフィー）/ むくれ / うろたえ

  ※エフェクト重ね（顔パーツと同層か上層）
    ├── 赤面（弱）
    ├── 赤面（強）
    └── 汗

【汚れパーツ】最上層、各変数が1のときに表示（0=透明1px画像）
  ├── f.if_chara_splashed   水浸し（③A選択で1）
  ├── f.if_chara_stomped    犬の足跡（⑥A選択で1）
  └── f.if_chara_stucked    葉・枝・ゴミ（3つめの不幸で1）
  ※3種は独立して重ねられる。全部乗る場合もある

【傘パーツ】ベースとは別素材（後ろ手に持つ）
  ├── 緑の傘
  ├── 緑の傘（壊れ）
  └── 赤い傘
```

汚れパーツは全身立ち絵・横歩き絵の両方に対応する。

## 魚座分岐の実装メモ

### 01_avan 星座選択後に以下を一括設定する

```ks
; 魚座分岐変数を一括設定
[eval exp="f.c_name       = (f.p_sign==12) ? 'アジサイ' : '＊'"]
[eval exp="f.ch_reflect   = (f.p_sign==12) ? 'true' : 'false'"]
[eval exp="f.rain_deg     = (f.p_sign==12) ? -5 : 5"]
[eval exp="f.walk_keyframe = (f.p_sign==12) ? 'walk_bg_scroll_rev' : 'walk_bg_scroll'"]
```

### 使用箇所

| 変数 | 使い方 |
|---|---|
| `f.c_name` | アジサイのセリフ直前に `#&f.c_name` と書く（通常の `#ajisai_n` は使わない） |
| `f.ch_reflect` | `[chara_show name="ajisai_n" layer="3" reflect="&f.ch_reflect"]` |
| `f.rain_deg` | `[show_rain]` の実装内でdeg値に `&f.rain_deg` を渡す |
| `f.walk_keyframe` | `[xanim name="walk_bg_loop" keyframe="&f.walk_keyframe" ...]` |

### macro.ks に追加が必要なもの

- `walk_bg_scroll_rev` キーフレーム（x=0 → x=2560 の逆スクロール）
- `show_rain` マクロをdeg変数対応に変更
